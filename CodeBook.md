# Code Book

## Variables description

* Subject
    + The subject ID
* Activity
    + The activity label. Possible values are:
        + WALKING
        + WALKING_UPSTAIRS
        + WALKING_DOWNSTAIRS
        + SITTING
        + STANDING
        + LAYING

Then for all the following variables, we display the mean of the values per subject/activity.
